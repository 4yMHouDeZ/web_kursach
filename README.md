# Сайт на тему "История Российской империи"
Курсовая работа по web-дизайну.

## index.html
На странице представлено: 
- HTML форма (Логин, Пароль, Почта);
- API-функция (storage для сохранения данных, введенных пользователем);
- Использование JavaScript возможностей.

## praviteli.html
На странице представлено:
- Список (с блоками информации о императорах и императрицах);
- Взаимодействие с DOM (Поиск правителя по имени).

## sobitiya.html
На странице представлено:
- Таблица;

## simvolika.html
На странице представлено:
- Видео;
- Аудио;
- API-функия audio;

# Итог
Всего 3 информативных страницы, (4ая не имеет отношения к теме задания).
Верстка на всех страницах выполнена по правилам семантики.
Сайт имеет мобильную версию.
